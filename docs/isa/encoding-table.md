## 解码优先级说明

- `opcode = 0x00`：按 `funct` 区分 R 格式 ALU 指令
- `opcode = 0x08`：按 `funct` 区分 `JR`/`JALR`
- 其余 opcode：仅由 opcode 唯一确定
